You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply? -- multiples two numbers -- 3.86.245.200/multiply?num1=5&num2=6
  • /github? -- pulls a query from request then runs it through github's rest api -- 3.86.245.200/github?query=users/amehlhase316/repos
  • /cipher? -- shifts characters of a given string by a given number -- 3.86.245.200/cipher?secret=xyz&shift=3 -- returns abc
  • /diceroll? -- takes two positive integers, one to set dice max value and amount rolled then returns the total value of the dice together -- 3.86.245.200/diceroll?dice=8&amount=5
